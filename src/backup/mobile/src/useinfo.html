주문방법
1. 홈페이지에 로그인하시고 원하시는 상품을 클릭하세요.
 (로그인하지 않고 비회원으로도 주문 가능합니다.)
2. '한 가지 상품'만 주문하실 경우 [즉시구매] 를 클릭합니다.
3. '여러 상품'을 주문하실 경우 [금고에 담기] 를 클릭합니다.
4. '나의 금고'에서 주문상품 확인 후 [주문하기] 를 클릭하세요.
5. 주문서 작성 후 하단의 [결제하기] 를 누르시면 주문완료됩니다.

* 위 방법이 어려우실 경우 온라인사업부(031-516-7706)로 연락주시면 전화주문도 가능하십니다.
