    구매시 사용 가능

쿠폰 내역이 없습니다.

  • 주문 할인쿠폰 적용방법
    1. ㆍ주문서 쿠폰 선택에서 고객님이 보유하신 [쿠폰번호]를 선택하시면 할인금액(또는 적립금액)이 나타납니다.
    2. ㆍ정률할인(적립)의 경우, 할인율(적립률)이 나타납니다.
  • 쿠폰 사용시 유의사항
    1. ㆍ쿠폰마다 사용 가능 금액과 사용기한이 정해져 있으며, 1개 주문서에 1개의 쿠폰만을 사용하실 수 있습니다.
    2. ㆍ쿠폰을 사용한 주문 후 반품/환불/취소의 경우 한번 사용하신 쿠폰은 다시 사용하실 수 없습니다.
    3. ㆍ쿠폰 적용 품목 및 카테고리가 제한된 경우 해당 품목 또는 카테고리에서만 쿠폰을 사용하실 수 있습니다.
    4. ㆍ할인/멤버십포인트 쿠폰은 실제 결제금액에 적용됩니다.
    5. ㆍ 모바일 아이콘 표시 쿠폰은 모바일에서만 사용이 가능합니다.
